Python

Python es uno de los lenguajes de programación más usados actualmentey la tendencia sigue al alza. Lo tiene todo: es de código abierto, una sintaxis sencilla y fácil de entender, por lo que ahorra tiempo y recursos. Es uno de los mejores para iniciarse en el mundo de la programación.

Java

Java es un lenguaje de programación de propósito general, orientado a objetos y diseñado para tener las dependencias de implementación mínimas posibles. Con este lenguaje de programación se pueden crear aplicaciones y procesos en múltiples dispositivos.

JavaScript

Un JavaScript es un lenguaje de programación interpretado, orientado a objetos y que se utiliza principalmente en la forma del lado del cliente.
